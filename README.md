如何取得拓元登入後的 SID（Wiindows電腦版本, 手機/平板不適用)

1. 第一次使用拓元需要註冊，通常搭配您的 Google 帳號即可, 且請在拓元輸入好帳號所需的身分證字號與電話綁定等等, 之後的登入即可不需再輸入，無綁定手機之帳號無法買票。
2. 正常情況下請在登入拓元購票後，會記住上次的登陸帳號，為確保 cookie 可以最大新鮮度，請先登出後在登入一次拓元，確保 cookie 約可保持至少約3~7天內不會過期，不需重新登入。
4. cookie SID 取得方式

A. 選 chrome 瀏覽器右上角 -> 更多工具 -> 開發人員工具

<img src="https://github.com/user-attachments/assets/76266f92-8679-4522-8b3b-5385d48520c7" width="50%" style="border: 2px solid black;" />

B. 選 Application (若沒看到請按右邊箭頭才會跑出來) -> storte 中的 cookies 並點 tixcraft.com -> 找到大寫 "SID" -> 看到 cookie Value 把下面這串 akt0vdu8uk44tlbtif0t559u6u 拷貝(ctrl+c)出來即可.

<img src="https://github.com/user-attachments/assets/b8b77755-3815-4ffd-9060-39804613692c" width="50%" style="border: 2px solid black;" />

4. 此時您的網頁仍可在拓元買票等等，但切記不要登出，否則 cookie SID 將會被清除，需要重新登入後提供新的給我。
5. 由於採用程式刷票，存在拓元鎖帳號的風險(我已採用較低風險)，需要自行評估風險。
6. 帳號被鎖後無法恢復，此身分證字號不可再度使用，實名制通常會另外填寫，所以沒差，電話號碼不可重複註冊，這個就再去找另外一個號碼，也不至於會有太大的問題。
